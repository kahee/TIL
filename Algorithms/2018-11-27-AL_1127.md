## Find Digits
[문제링크](https://www.hackerrank.com/challenges/find-digits/problem)

## 문제 풀이
```python
#!/bin/python3

import math
import os
import random
import re
import sys

# Complete the findDigits function below.
def findDigits(n):

    result = 0
    for num in str(n):
        if int(num) == 0:
            continue
        if n%int(num)==0:
            result +=1

    return result


if __name__ == '__main__':
    fptr = open(os.environ['OUTPUT_PATH'], 'w')

    t = int(input())

    for t_itr in range(t):
        n = int(input())

        result = findDigits(n)

        fptr.write(str(result) + '\n')

    fptr.close()

```
